# 模块加载系统概述

## CommonJS

+ 模块加载完成会缓存加载，如需再次运行模块需要清除缓存
+ 模块加载完成前会阻塞代码运行（同步加载）
+ 环境：服务器，是Nodejs的模块规范

## AMD

+ 异步加载

+ 管理模块之间的依赖性

+ 环境：浏览器，requireJS是参照AMD规范实现的

  

## UMD

+ 兼容commonjs和AMD的同时，还兼容全局引入

+ 环境：浏览器或服务器

  

## ES Module

+ 按需加载（编译时加载）
+ import和export只能在最顶层，import语句可以在代码块中实现异步动态按需加载
+ 环境：浏览器或Node后端服务，是ES6的最新语法支持规范
